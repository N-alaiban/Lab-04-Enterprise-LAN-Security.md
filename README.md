# WBMUD Enterprise LAN Security Assessment

This repository contains my work for Lab 04 in IT 520.

In this project, I designed a segmented enterprise network for a fictional water utility system (WBMUD). The network separates IT, OT, and Remote environments to improve security and reduce risks. I also included a DMZ attacker node to simulate an external threat and test how the network responds to attacks.

The lab focuses on applying the OSI model, configuring IP addressing and routing, and analyzing protocol security. I compared TCP and UDP, explained the difference between HTTP and HTTPS, and demonstrated why SSH is more secure than Telnet.

In addition, I explored a real-world vulnerability (Shellshock) and explained how it could be used by an attacker to gain access to the network. I also described the attack path, identified the root cause, and suggested solutions to prevent similar attacks in the future.

Overall, this lab helped me understand how network design and cybersecurity work together to protect critical infrastructure systems.
